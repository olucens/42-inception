1. What 3 kernel features make containers possible?
Bu unarchive image of operation system(linux) we able to change the root of file system, maybe only for one process, by using comand:

```sh
chroot <where installed OS> /bin/sh
```

then we still have to unmound namespaces to provide isolation for future container, and for that we need to do before start in container that 

```sh
sudo unshare \
  --pid \
  --fork \
  --mount-proc \
  --net \
  --uts \
  --ipc \
  chroot /tmp/mycontainer /bin/sh
```
After those steps we will have something close to docker container, but it's also setted up network, limist and etc.
